# Component Inventory

## Form Components (src/components/forms/)
Multi-step event creation form components. Each step is a separate component.

## Event Components (src/components/events/)
Display components for event cards, lists, and detail views.

## Global Components (src/components/global/)
Layout components: header, footer, navigation, sidebar.

## Preview Components (src/components/preview/)
Mobile app preview mockup shown on the website:
- `AppPreview.tsx` — State machine: home → results → detail
- `PhoneFrame.tsx` — Phone bezel with status bar
- `constants.ts` — Design tokens matching Expo app
- `mockEvents.ts` — 63 realistic Swedish events
- `screens/HomeScreen.tsx` — Category tiles, tags, Go.Do! button
- `screens/ResultsScreen.tsx` — Map/list view, subcategory chips
- `screens/EventDetailScreen.tsx` — Hero gradient, content sheet

## UI Components (src/components/ui/)
shadcn/ui base components (Button, Dialog, Select, etc.).
Do NOT modify these directly — they are generated by shadcn CLI.
To add new ones: `npx shadcn@latest add <component>`
